================================================================================
                    ENDPOINTS DE REPORTES PDF - MFarm
================================================================================
Este archivo contiene todos los endpoints necesarios para generar reportes PDF.
Cada entrada incluye: ruta del endpoint, datos que se envian y datos que debe
retornar el backend en formato PDF.

================================================================================
PRODUCCION
================================================================================

2. Reporte de Inseminaciones y Partos
   GET /reports/production/inseminations-births/pdf/:farm_id?start_date=YYYY-MM-DD&end_date=YYYY-MM-DD&orientation=portrait&format=A4
   Recibe:
     - farm_id (param): string
     - start_date (query): string (YYYY-MM-DD)
     - end_date (query): string (YYYY-MM-DD)
     - orientation (query): string (portrait|landscape)
     - format (query): string (A4)
   Debe retornar (PDF con):
     - Tabla de inseminaciones: fecha, cerda, berraco/muestra, tecnico, resultado
     - Tabla de partos: fecha, cerda, nacidos vivos, nacidos muertos, momificados
     - Resumen: total inseminaciones, total partos, efectividad

3. Reporte de Grupos
   GET /reports/production/groups/pdf/:farm_id?start_date=YYYY-MM-DD&end_date=YYYY-MM-DD&orientation=landscape&format=A4&group_id=optional
   Recibe:
     - farm_id (param): string
     - start_date (query): string (YYYY-MM-DD)
     - end_date (query): string (YYYY-MM-DD)
     - orientation (query): string (portrait|landscape)
     - format (query): string (A4)
     - group_id (query, opcional): string
   Debe retornar (PDF con):
     - Tabla de grupos: nombre, etapa, cantidad cerdos, peso promedio,
       fecha entrada, dias en etapa
     - Movimientos por grupo: fecha, tipo movimiento, cantidad, origen, destino
     - Resumen productivo: entradas, salidas, muertes, peso, consumo

4. Reporte de Mortalidad
   GET /reports/production/mortality/pdf/:farm_id?start_date=YYYY-MM-DD&end_date=YYYY-MM-DD&orientation=portrait&format=A4
   Recibe:
     - farm_id (param): string
     - start_date (query): string (YYYY-MM-DD)
     - end_date (query): string (YYYY-MM-DD)

6. Reporte de Rendimiento Reproductivo
   GET /reports/production/reproductive/pdf/:farm_id?start_date=YYYY-MM-DD&end_date=YYYY-MM-DD&orientation=portrait&format=A4
   Recibe:
     - farm_id (param): string

11. Reporte de Compras y Cuentas por Pagar

   Recibe:
     - farm_id (param): string

     - end_date (query): string (YYYY-MM-DD)
     - orientation (query): string (portrait|landscape)
     - format (query): string (A4)
   Debe retornar (PDF con):
     - Tabla compras: fecha, proveedor, producto, cantidad, precio unitario, total, factura

12. Reporte de Analisis de Costos
   GET /reports/finance/costs/pdf/:farm_id?start_date=YYYY-MM-DD&end_date=YYYY-MM-DD&orientation=landscape&format=A4
   Recibe:
     - farm_id (param): string
     - start_date (query): string (YYYY-MM-DD)
     - end_date (query): string (YYYY-MM-DD)
     - orientation (query): string (portrait|landscape)
     - format (query): string (A4)
   Debe retornar (PDF con):
     - Tabla costos por grupo: grupo, etapa, cerdos, alimento, medicamento, otros, total, costo/cerdo
     - Tabla costos por cerdo: cerdo, grupo, alimento, medicamento, otros, total, peso, costo/kg
     - Tabla costos por tipo: tipo, total, porcentaje, registros
     - Resumen: costo total, costo/grupo, costo/cerdo, costo/kg
